🚀 5-MINUTE QUICK START (LOCAL)

  # Terminal 1: Start Backend
  cd backend
  python3 -m venv venv
  source venv/bin/activate                  # Windows: venv\Scripts\activate
  pip install -r requirements.txt
  python3 main.py
  # → Running on http://0.0.0.0:8000

  # Terminal 2: Start Frontend
  cd frontend
  npm install
  npm run dev
  # → Running on http://localhost:5173

  # Terminal 3: Generate DNS Traffic
  cd scripts
  pip install dnspython
  python3 generate_dns_traffic.py

  # Terminal 4: Open Browser
  http://localhost:5173

  ✓ See live DNS queries, alerts, and risk scores!

🎓 HOW IT WORKS (Architecture)

  1. Network Traffic
     ↓
  2. Suricata Captures DNS (UDP:53)
     ↓
  3. Suricata Logs to eve.json
     ↓
  4. Backend reads eve.json (async file watcher)
     ↓
  5. DNSDetector analyzes each DNS event
     ↓
  6. Risk scoring & threat detection
     ↓
  7. Broadcasting via WebSocket
     ↓
  8. Frontend receives & displays real-time
     ↓
  9. Dashboard updates instantly
